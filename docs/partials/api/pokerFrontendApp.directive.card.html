card
directive in module pokerFrontendApp

Description

card

A card directive that displays a svg image of the given color (color + value). Can be flipped with css animation and comes in two different sizes, normal and mini.

Usage

as element:
<card
       flipped="{boolean}"
       color="{String}"
       sise="{String}">
</card>

Parameters

ParamTypeDetails
flippedboolean

indicates if the card is flipped. Can be changed in real time and the card changes too.

colorString

represents the value and the color of the card. E.g. 'clubs_3' represents a three of clubs.

siseString

represents the size of the card. If 'mini', the card has smaller size and an automatic mouseover zoom.
