cardZoom
directive in module pokerFrontendApp

Description

cardZoom

An extension of the card directive to zoom in on the card, if the size is 'mini'.

Usage

as attribute
<img card-zoom
     mini="{String}">
   ...
</img>

Parameters

ParamTypeDetails
miniString

If the size is 'mini' than a mouseenter/leave listener is registered to animate a zoom factor on the svg image of the card.
