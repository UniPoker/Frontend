MainCtrl
function in module pokerFrontendApp

Description

MainCtrl

Controller of the pokerFrontendApp

Usage

MainCtrl();

Methods

  • $on()

    list_rooms_response

    Registered handler for list_rooms_response. If this handler is triggered, the rooms Array in the response is set to the local rooms variable.

  • $on()

    create_room_response

    Registered handler for create_room_response. If this handler is triggered and the error code is 0, the new room is pushed into the local rooms variable and the view changes to game view. On error it displays a notification with the given error.

  • $on()

    join_room_response

    Registered handler for join_room_response. If the error code is 0 the room_id of the user is set and the view changes to the game view.

  • $on()

    new_rooms_notification

    Notification send from server. If this notification gets triggered, all rooms get updated.

  • add_room()

    Requests create_room from the server. A handler needs to be registered separately.

  • init()

    Function that is called initially on load of main route. If the user is in a room, a leave_room request is send to the server. Anyway list_rooms request is send to server.

  • join_room()

    Requests join_room from the server. A handler needs to be registered separately.

  • list_rooms()

    Requests list_rooms from the server. A handler needs to be registered separately.
